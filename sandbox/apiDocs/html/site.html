Site
Pass object to the site parameter to add to site property to request.
pbjs.setConfig({ site: { content: language: 'en' } } } });
Arguments:
  • site: object

    Adds support for setting site in the config, and the pbs adapter should add this value to oRTB requests.

Response
Response Description
{ response: example }
  • param1: string

    String indicating some value that has to be passed as a string type.

  • param2: number

    Number indicating some value that has to be passed as a number type.

    Note: This number should be between 0 and infinity.
